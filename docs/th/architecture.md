# เอกสารการออกแบบสถาปัตยกรรม FOFO Studio

**กำลังแปล** | กรุณาอ้างอิง[เวอร์ชันภาษาอังกฤษ](../en/architecture.md)สำหรับข้อมูลล่าสุด

---

## ภาพรวมสถาปัตยกรรมระบบ

FOFO Studio ใช้สถาปัตยกรรมไมโครเซอร์วิสที่แยก frontend-backend รวมบริการ blockchain และ AI เพื่อสร้างแพลตฟอร์ม Phygital

## สถานะเอกสาร

เอกสารนี้กำลังอยู่ในระหว่างการแปล สำหรับข้อมูลทั้งหมด กรุณาอ้างอิง[เอกสารการออกแบบสถาปัตยกรรมภาษาอังกฤษ](../en/architecture.md)

## ส่วนหลัก

1. **ภาพรวมสถาปัตยกรรมระบบ**
2. **สแต็กเทคโนโลยีโดยละเอียด**
3. **โครงสร้างพื้นฐาน Blockchain**
4. **สถาปัตยกรรมบริการ AI**
5. **สถาปัตยกรรมความปลอดภัย**
6. **การเพิ่มประสิทธิภาพ**
7. **การตรวจสอบและบันทึก**
8. **สถาปัตยกรรมการปรับใช้**
