# FOFO Studio Brand Guidelines

## Brand Core

### Brand Name

**FOFO Studio**

Full Name: **For Others, For Ourselves**

### Brand Slogans

- **Main Slogan**: "Trade Together, Win Alone"
- **Sub Slogan**: "Dress Like You Own The Chart"
- **Philosophy**: "Code for the Community. Style for the Self"

### Brand Positioning

A phygital platform integrating Web3, AIGC, and street culture, providing unique product experiences for creators and traders in the digital age.

### Target Audience

- Day Traders and Crypto native users
- Street culture enthusiasts
- AI creators
- Web3 early adopters

## Visual Identity System

### Logo Design Concept

The FOFO Logo should reflect the concept of "Duality":

- **First two letters "FO"**: Warm tones (red/orange), representing passionate "For Others"
- **Last two letters "FO"**: Cool tones (cyan/blue), representing calm and rational "For Ourselves"
- **Middle**: Glitch dividing line, constantly merging and separating

#### Logo Dynamic Effect

```
[FO]━━━━━━━━━━━━━━━━━━━━━━━━━━━━[FO]
 Warm         Glitch Line         Cool
(Red)      (Flicker/Merge)      (Cyan)
```

### Color System

#### Primary Colors

- **Dark Background**: #0A0A0A (near black)
- **Neon Cyan**: #00FFFF
- **Neon Magenta**: #FF00FF
- **Neon Pink**: #FF1493

#### Secondary Colors

- **Success Green**: #00FF00 (Green Candles)
- **Warning Orange**: #FFA500
- **Error Red**: #FF0000 (Red Candles)
- **Neutral Gray**: #808080

#### Color Usage Guidelines

- **Background**: Dark mode primary, creating Cyberpunk atmosphere
- **Emphasis**: Use neon colors to highlight important elements
- **Text**: White or light gray, ensuring readability
- **Links/Buttons**: Neon cyan or magenta with glow effects

### Typography System

#### Primary Font (Headings)

- **Names**: "Cyberpunk" / "Orbitron" / "Rajdhani"
- **Characteristics**: Bold, futuristic, sans-serif
- **Usage**: Logo, large headings, button text

#### Code Font (UI Elements)

- **Names**: "Fira Code" / "JetBrains Mono" / "Courier New"
- **Characteristics**: Monospace, terminal style
- **Usage**: Prompt input boxes, data panels, code display

#### Body Font

- **Names**: "Inter" / "Roboto" / "SF Pro"
- **Characteristics**: Clear and readable, modern
- **Usage**: Body content, product descriptions

### Icon System

#### Icon Style

- **Pixelated**: Retro pixel style
- **Glitch Effect**: Glitch art style
- **Neon Glow**: Glowing edge effects

#### Core Icons

- **FOFO Logo**: Main brand identifier
- **AI Generation**: Neural network icon
- **NFT**: Diamond + chain
- **Community**: Multiple user avatars
- **Trading**: K-line chart icon

### Graphic Elements

#### Glitch Effect

- **Usage**: Transition animations, loading effects, emphasis elements
- **Implementation**: CSS animation + SVG filter

#### Pixelation Effect

- **Usage**: Retro style decoration, background patterns
- **Implementation**: Low-resolution images, pixel grid

#### K-line Chart Elements

- **Usage**: Decorative background, data visualization
- **Style**: Minimalist lines, neon color highlights

## Brand Tone

### Core Traits

- **Confident**: Not humble, directly express opinions
- **Humorous**: Play with memes, self-deprecating, dark humor
- **Technical**: Use industry terminology, but not overly esoteric
- **Rebellious**: Challenge traditions, break rules
- **Community-oriented**: Emphasize "we" rather than "I"

### Tone Examples

#### ✅ Correct Examples

- "Mint your first NFT, then wear it on your body."
- "Survive the crash, shine in the bull market."
- "This is not a piece of clothing, this is your digital identity."
- "Share Alpha, earn Karma. WAGMI."

#### ❌ Incorrect Examples

- "We provide high-quality products and services." (Too formal)
- "Welcome to purchase our products." (Too traditional)
- "We are the most professional platform." (Too commercial)

### Common Terminology

- **Alpha**: Trading intelligence
- **Bag**: Holdings
- **Candle**: K-line
- **Mint**: Create NFT
- **Drop**: Limited release
- **WAGMI**: We All Gonna Make It
- **NGMI**: Not Gonna Make It
- **Diamond Hands**: Hold firmly
- **Paper Hands**: Sell easily
