# FOFO Studio 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品定位

FOFO Studio 是一个融合 Web3、AIGC 与潮流文化的虚实结合平台，核心功能是让用户通过 AI 生成独特的 NFT 图案，并将其打印在实体商品上。

### 1.2 目标用户

#### 主要用户群体

1. **Day Trader** (日间交易者)
   - 年龄: 25-35 岁
   - 特征: 追求个性化、了解 Crypto 文化、有购买力
   - 需求: 独特的自我表达、交易文化的身份认同

2. **Crypto 原生用户**
   - 年龄: 20-40 岁
   - 特征: 深度参与 Web3、持有 NFT、熟悉区块链
   - 需求: 虚实结合的产品体验、DAO 参与感

3. **潮流文化爱好者**
   - 年龄: 18-30 岁
   - 特征: 喜欢 Meme、Cyberpunk、Retro 风格
   - 需求: 限量版产品、社区归属感

4. **AI 创作者**
   - 年龄: 20-35 岁
   - 特征: 使用 AI 工具创作、希望变现创意
   - 需求: 创作平台、版税收入

### 1.3 核心价值主张

- **Prompt to Product**: 一句话生成独一无二的产品
- **数字所有权**: NFT 确权，真正拥有你的设计
- **社区共创**: 分享创意，获得版税收益
- **虚实融合**: 数字创作立即实体化

## 2. 功能需求

### 2.1 The Neural Mint (神经铸造局)

#### 功能描述

用户通过输入 Prompt，AI 生成图像，自动铸造为 NFT，并打印在实体商品上。

#### 用户流程

1. **入口**: 点击 "Create" 或 "The Neural Mint"
2. **输入 Prompt**: 
   - 文本输入框（终端风格）
   - 风格芯片快捷按钮
   - Prompt 示例库
3. **生成预览**: 
   - 显示 4 张变体
   - 加载动画（代码解密效果）
4. **选择与编辑**:
   - 选择最喜欢的一张
   - 可选：调整参数重新生成
5. **选择载体**:
   - T恤、卫衣、手机壳、鼠标垫等
   - 实时 3D 预览
   - AR 试穿（移动端）
6. **铸造 NFT**:
   - 连接钱包
   - 确认 Gas 费用
   - 铸造完成
7. **下单**:
   - 选择尺寸、颜色
   - 填写地址
   - 支付（Crypto 或法币）
8. **追踪**:
   - 订单状态
   - 生产进度
   - 物流信息

#### 技术要求

- **响应时间**: 图像生成 < 30 秒
- **并发处理**: 支持 100+ 并发请求
- **NFT 铸造**: < 5 秒（Polygon 网络）
- **预览加载**: < 2 秒

#### 业务规则

- 免费用户: 每天 3 次免费生成
- 付费用户: 无限生成 + 更高质量
- NFT 铸造: 需支付 Gas 费用（约 $0.01-0.1）

### 2.2 The Market (交易大厅)

#### 功能描述

展示所有产品，采用独特的分类方式和交易风格的 UI。

#### 核心功能

1. **产品展示**:
   - 分类浏览（Green Candles, Internet Heritage, Neural Link）
   - 搜索功能
   - 筛选（价格、风格、载体）

2. **产品详情页**:
   - 图片展示（支持 360 度旋转）
   - 数据面板（模仿彭博终端）
   - 设计灵感说明
   - 购买按钮（"LONG" 按钮）
   - 关联推荐（"Hedging Options"）

3. **购物流程**:
   - 购物车（"Portfolio"）
   - 结算页面
   - 支付方式选择（Crypto / 信用卡）
   - 订单确认

#### UI 特色

- **跑马灯**: 顶部滚动显示 Crypto 价格、热销产品
- **动态定价**: 部分产品价格与 BTC 价格挂钩
- **FOMO 倒计时**: 限量版产品倒计时抢购

### 2.3 Alpha DAO (情报局)

#### 功能描述

社区内容分享平台，用户分享交易心得、Meme、潮流资讯。

#### 核心功能

1. **内容发布**:
   - 文字帖子
   - 图片分享
   - 交易截图
   - Meme 图

2. **互动功能**:
   - 点赞、评论、分享
   - 收藏（加入个人 Alpha 库）
   - 举报不当内容

3. **积分系统**:
   - Karma Points（因果积分）
   - 发布内容获得积分
   - 高质量内容额外奖励
   - 积分兑换产品/折扣

4. **内容推荐**:
   - 热门内容
   - 关注列表
   - 个性化推荐

#### 业务规则

- 每日发帖限制: 防止 spam
- 内容审核: AI + 人工审核
- 积分有效期: 6 个月

### 2.4 Co-Create Lab (共创实验室)

#### 功能描述

开放设计资源，允许用户二次创作，被选中后可量产并获得版税。

#### 核心功能

1. **资源下载**:
   - Logo 源文件
   - 设计模板
   - 图案素材

2. **上传作品**:
   - 上传设计图
   - 填写设计说明
   - 选择授权方式

3. **社区投票**:
   - 用户投票选择优秀设计
   - 设计师获得版税分成（10-30%）

4. **版税追踪**:
   - 显示设计被使用次数
   - 版税收入统计
   - 自动分发收益

### 2.5 The Vault (私人金库)

#### 功能描述

VIP 会员专区，提供独家产品和高级定制服务。

#### 准入条件

- 持有 FOFO NFT
- 消费满 $1000
- 持有治理代币（未来）

#### 特权功能

- 独家产品购买权
- 高级定制服务
- 优先购买限量版
- 专属客服通道

### 2.6 Custom Loadout (自定义装备)

#### 功能描述

高级定制服务，允许用户印上个人专属内容。

#### 可定制内容

- 交易单截图
- 钱包地址二维码
- 个人 Meme 形象
- AI 生成的个人肖像
- 自定义文字

#### 流程

1. 上传/选择内容
2. AI 处理优化
3. 预览效果
4. 确认下单
5. 生产发货

## 3. 非功能性需求

### 3.1 性能要求

- **页面加载**: < 2 秒
- **API 响应**: < 500ms (P95)
- **图像生成**: < 30 秒
- **支付处理**: < 10 秒

### 3.2 可用性要求

- **系统可用性**: 99.9% (年度停机时间 < 8.76 小时)
- **数据备份**: 每日自动备份
- **故障恢复**: < 1 小时

### 3.3 安全要求

- **数据加密**: HTTPS + 数据库加密
- **用户认证**: Web3 钱包 + 传统认证
- **支付安全**: PCI-DSS 合规
- **智能合约审计**: 第三方安全审计

### 3.4 兼容性要求

- **浏览器**: Chrome, Firefox, Safari, Edge (最新 2 个版本)
- **移动端**: iOS 14+, Android 10+
- **钱包**: MetaMask, WalletConnect 兼容钱包

### 3.5 可扩展性要求

- **用户规模**: 支持 10 万+ 注册用户
- **并发用户**: 支持 1000+ 并发在线
- **存储**: 支持 PB 级图像存储

## 4. UI/UX 设计要求

### 4.1 视觉风格

- **主题**: Cyberpunk + Retro + Glitch
- **颜色**: 深色背景 + 霓虹色（青色、紫色、粉色）
- **字体**: 粗犷无衬线 + 终端代码字体
- **动效**: Glitch 故障效果 + 像素化动画

### 4.2 交互设计

- **鼠标悬停**: K 线图动画、像素化图标
- **加载动画**: 代码解密、矩阵数字雨
- **页面切换**: 故障过渡效果

### 4.3 响应式设计

- **桌面端**: 1920x1080 为主
- **平板端**: 768x1024
- **移动端**: 375x667 起

- **用户指标**: DAU, MAU, 留存率
- **交易指标**: GMV, 订单数, 客单价

- **社区指标**: 内容发布数, 互动率

### 5.2 用户行为分析

- 用户路径追踪

- 转化漏斗分析
- A/B 测试框架
