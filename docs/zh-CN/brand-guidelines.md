# FOFO Studio 品牌指南

## 品牌核心

### 品牌名称

**FOFO Studio**

全称: **For Others, For Ourselves**

### 品牌口号

- **主口号**: "Trade Together, Win Alone" (一起交易，独自赢利)
- **副口号**: "Dress Like You Own The Chart" (穿得像你掌控K线图一样)
- **理念**: "Code for the Community. Style for the Self" (为社区编码，为自我型格)

### 品牌定位

一个融合 Web3、AIGC 与潮流文化的虚实结合平台，为数字时代的创作者和交易者提供独一无二的产品体验。

### 目标受众

- Day Trader 和 Crypto 原生用户
- 潮流文化爱好者
- AI 创作者
- Web3 早期 adopters

## 视觉识别系统

### Logo 设计理念

FOFO Logo 应该体现"二元性"（Duality）的概念：

- **前两个字母 "FO"**: 暖色调（红色/橙色），代表热情的 "For Others"
- **后两个字母 "FO"**: 冷色调（青色/蓝色），代表冷静理性的 "For Ourselves"
- **中间**: Glitch（故障）分割线，不断融合又分离

#### Logo 动态效果

```
[FO]━━━━━━━━━━━━━━━━━━━━━━━━━━━━[FO]
 暖色         故障分割线         冷色
(红色)      (闪烁/融合)        (青色)
```

### 色彩系统

#### 主色调

- **深色背景**: #0A0A0A (接近黑色)
- **霓虹青**: #00FFFF (Cyan)
- **霓虹紫**: #FF00FF (Magenta)
- **霓虹粉**: #FF1493 (Deep Pink)

#### 辅助色

- **成功绿**: #00FF00 (Green Candles)
- **警告橙**: #FFA500 (Orange)
- **错误红**: #FF0000 (Red Candles)
- **中性灰**: #808080 (Gray)

#### 色彩使用规范

- **背景**: 深色模式为主，营造 Cyberpunk 氛围
- **强调**: 使用霓虹色突出重要元素
- **文本**: 白色或浅灰色，确保可读性
- **链接/按钮**: 霓虹青色或紫色，带发光效果

### 字体系统

#### 主字体（标题）

- **名称**: "Cyberpunk" / "Orbitron" / "Rajdhani"
- **特征**: 粗犷、未来感、无衬线
- **用途**: Logo、大标题、按钮文字

#### 代码字体（UI 元素）

- **名称**: "Fira Code" / "JetBrains Mono" / "Courier New"
- **特征**: 等宽、终端风格
- **用途**: Prompt 输入框、数据面板、代码显示

#### 正文字体

- **名称**: "Inter" / "Roboto" / "SF Pro"
- **特征**: 清晰易读、现代
- **用途**: 正文内容、产品描述

### 图标系统

#### 图标风格

- **像素化**: 复古像素风格
- **Glitch 效果**: 故障艺术风格
- **霓虹发光**: 带发光边缘效果

#### 核心图标

- **FOFO Logo**: 品牌主标识
- **AI 生成**: 神经网络图标
- **NFT**: 菱形 + 链条
- **社区**: 多人头像
- **交易**: K 线图图标

### 图形元素

#### Glitch 效果

- **用途**: 过渡动画、加载效果、强调元素
- **实现**: CSS animation + SVG filter

#### 像素化效果

- **用途**: 复古风格装饰、背景图案
- **实现**: 低分辨率图像、像素网格

#### K 线图元素

- **用途**: 装饰背景、数据可视化
- **风格**: 简约线条、霓虹色高亮

## 品牌语调

### 核心特质

- **自信**: 不谦虚，直接表达观点
- **幽默**: 玩梗、自嘲、黑色幽默
- **技术感**: 使用行业术语，但不过于深奥
- **反叛**: 挑战传统，打破规则
- **社区感**: 强调"我们"而非"我"

### 语调示例

#### ✅ 正确示例

- "Mint 你的第一个 NFT，然后把它穿在身上。"
- "在崩盘中幸存，在牛市中闪耀。"
- "这不是一件衣服，这是你的数字身份。"
- "分享 Alpha，获得 Karma。WAGMI。"

#### ❌ 错误示例

- "我们提供高质量的产品和服务。" (太正式)
- "欢迎购买我们的产品。" (太传统)
- "我们是最专业的平台。" (太商业)

### 常用术语

- **Alpha**: 交易情报
- **Bag**: 持仓
- **Candle**: K 线
- **Mint**: 铸造 NFT
- **Drop**: 限量发售
- **WAGMI**: We All Gonna Make It
- **NGMI**: Not Gonna Make It
- **Diamond Hands**: 坚定持有
- **Paper Hands**: 轻易抛售
